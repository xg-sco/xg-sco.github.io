minio

介绍

MinIO是根据GNU Affero通用公共许可证v3.0发布的高性能对象存储。它与Amazon S3云存储服务兼容的API。它可以处理非结构化数据，例如照片，视频，日志文件，备份和容器图像，当前最大支持对象大小为5TB。

下载

下载地址:

运行

创建四个文件, 每一个文件表示一个minio

然后访问: http://localhost:9000/

账号密码都是minioadmin

创建桶

Create Bucket

随便上传一个文件

本地的文件系统就会存储这个上传的文件

当文件系统小于 N/2 (一半)时, 此时就不能上传文件了

此时是有4个文件系统, 我们删掉3个

这个时候就登不上了minio了, 也不能接着上传文件了
