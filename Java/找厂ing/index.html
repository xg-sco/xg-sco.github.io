下班后不接受任何工作的电话微信以及任何能够找到我的方式可以吗？

1不应该填详细信息，但是我目前遇到的所有公司都要先填信息，而且还是比较靠谱的才填
2待遇好的公司会瓶装水，我真正工作的公司里就直接给一次性杯子或茶杯，这样的公司在我看来可能更实在。
3好公司都是用大理石，而不是贴墙纸。这个是的，但是主要还是看氛围，再好的环境也是可能不太行。
4卫生间要好。 是的，但是和第三条一样，再好的环境可能也只是办公楼建设好，但是完不完善，还是要自己亲身去感受
5工位环境大好，不要打鸡血横幅
6福利待遇不说，连基本的劳动合同等都不知道的立马走
7要你交方案的不行，白嫖你的经验
8五险一金和工资发放具体时间不在次月的15日之前都是有问题，
